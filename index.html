Projetos em Destaque

Contracto - Gerenciador de Contratos

Uma aplicação web desenvolvida para gerenciar contratos, com foco na segurança e organização dos dados. O backend foi construído para lidar com:

  • Sistema de autenticação JWT (login/registro) e autorização com níveis de acesso.
  • Integração robusta com banco de dados MySQL para persistência de informações de usuários e arquivos.
  • Armazenamento seguro e gerenciamento de arquivos de contrato enviados pelos usuários.
